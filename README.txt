Woo Braintree Payment Gateway allows you to accept payments using credit cards on your WooCommerce store.


== Description ==
Woo Braintree Payment Gateway authorizes credit card payments and processes them securely with your merchant account.

= Plugin Functionality: =

* Easy to install and configure.
* Compatible with WordPress/Woocommerce plugins
* You don’t need any extra plugins or scripts to process the Transaction
* Accepts all major credit cards
* Refunds functionality available.
* Tested this plugin on the Braintree sandbox (test) servers to ensure your customers don't have problems paying you.
* Our plugin handles warnings/errors correctly thrown in response so that user do not have to worry about dealing with those types of situations.
* Log write functionality available for transactions.

We always welcomes user suggestions. Let us know what you think about this plugin you liked or may have disliked. Users feedback is important for us to improve more our plugin.

== Installation ==

= Minimum Requirements =

* WooCommerce 2.1 or higher

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Woo Braintree Payment Gateway, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type Woo Braintree Payment Gateway and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.
